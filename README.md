# BlockAttend

BlockAttend is a secure attendance management system built with React, Node.js, Express, and PostgreSQL. The platform enables attendance tracking, role-based authentication, attendance analytics, AI-powered risk alerts, and SHA-256 hashing for attendance record integrity.

## Features

* Secure JWT Authentication
* Role-Based Access Control
* Attendance Management
* Attendance Analytics Dashboard
* AI Risk Alerts
* SHA-256 Record Hashing
* PostgreSQL Database
